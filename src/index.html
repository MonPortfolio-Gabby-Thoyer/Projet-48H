DDe nos jours, les clés USB et les disques durs sont des dispositifs couramment utilisés pour stocker et transporter des données importantes. Cependant, leur portabilité et leur commodité les rendent également vulnérables aux risques cyber. Les cybercriminels peuvent facilement exploiter les vulnérabilités de ces dispositifs pour voler des informations confidentielles ou propager des logiciels malveillants. Voici 5 bonnes pratiques à adopter pour garder en sécurité vos données :

ÉVITEZ D'INSÉRER UNE CLÉ USB OU UN DISQUE DUR DE SOURCE INCONNUE

Les clés USB et les disques durs provenant de sources non fiables peuvent contenir des virus, des logiciels malveillants, des rançongiciels... Ces sources non fiables peuvent être : un parking, dans une salle de classe, donnée à un salon de l’étudiant… Il est préférable de les éviter autant que possible et de se limiter aux sources connues et fiables. Vous pouvez trouver facilement des clés USB à bas prix dans un supermarché ou sur Amazon. Ne laissez personne insérer une clé USB ou un disque dur dans votre PC.

UTILISEZ UN LOGICIEL ANTIVIRUS POUR SCANNER LES CLÉS USB ET LES DISQUES DURS AVANT DE LES UTILISER

Les virus, les logiciels malveillants, les rançongiciels peuvent se propager facilement via les clés USB et les disques durs. Il est donc important de s'assurer que tous les fichiers sont sûrs avant de les ouvrir ou de les transférer sur votre ordinateur. Il existe des antivirus gratuits et pratiques comme Avast, Kaspersky, AVG… qui protègent votre PC et vont scanner les clés USB que vous branchez sur votre ordinateur.

ÉVITEZ DE PRÊTER VOTRE CLÉ USB OU VOTRE DISQUE DUR

Une personne peut utiliser votre clé pour stocker des virus, des logiciels malveillants, des rançongiciels ou encore des logiciels piratés. Ces logiciels malveillants peuvent aussi se déplacer d’eux-mêmes sur la clé si l’ordinateur est infecté sans le consentement de la personne. Il est illégal de détenir des copies piratées de logiciels.

EVITEZ D’UTILISER UNE CLÉ USB COMME STOCKAGE PRINCIPAL

Les clés USB peuvent être pratiques pour le stockage de fichiers, mais elles ne sont pas aussi fiables que les disques durs ou les serveurs de stockage en ligne. Il est donc important d'avoir une sauvegarde sur un autre support. Vous pouvez utiliser OneDrive, Google Drive, pCloud, iCloud si vous avez Iphone/Mac, pour vos cours et vos fichiers que vous ne souhaitez pas perdre ou que vous souhaitez accéder partout.

EVITEZ D’UTILISER UNE MÊME CLÉ USB OU DISQUE DUR POUR LE PERSONNEL ET LE PROFESSIONNEL

Pour éviter qu’une clé USB ou qu’un disque dur se connecte à plusieurs ordinateurs différents qui n’ont pas les mêmes sécurités, il est conseillé d’utiliser un support de stockage pour votre usage personnel et un support de stockage pour votre usage professionnel.
