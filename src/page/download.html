Sensibilisation : Téléchargement

Le téléchargement de logiciels ou d’applications peut présenter des risques tels que l'installation virus ou logiciels malveillants. Les logiciels ou applications malveillants qui peuvent ralentir ou prendre possession de votre ordinateur ou votre téléphone, ils peuvent récupérer aussi les données dessus autant les fichiers que les mots de passe... Il faut donc être vigilant.

Voici 5 bonnes pratiques à adopter pour télécharger en toute sécurité :

TÉLÉCHARGEZ SUR LE SITE DE L'ÉDITEUR

Il est important de télécharger un logiciel sur le site officiel de l’éditeur pour risquer qu’une tierce personne ait ajouté un composant malveillant dans le l’installateur du logiciel. N’hésitez pas à chercher sur votre moteur de recherche le nom du logiciel que vous souhaitez télécharger pour trouver l’éditeur, celui qui a créé le logiciel.

SCANNER L’INSTALLATEUR

Quand vous cliquez sur le télécharger, il va télécharger un installateur du logiciel que vous souhaitez. Avant de double cliquer dessus pour le lancer, vous pouvez le scanner avec votre antivirus pour qu’il vérifie qu’il n’y a rien de suspect dedans. Il est important d’avoir son antivirus à jour avant de lancer le scan, à chaque mise à jour votre antivirus “apprend” à reconnaître les nouveaux virus et autres logiciels malveillants.

VÉRIFIEZ LES AVIS

Avant de télécharger un logiciel ou une application, regardez les avis sur différents sites pour savoir s’il contient des publicités, des logiciels tierces, un potentiel virus, etc… Vous pouvez trouver différents sites d’avis pour des applications de téléphone ou directement sur le Store d’applications de votre téléphone. C’est aussi important quand vous téléchargez un plugin sur votre moteur de recherche.

FAITES ATTENTION À L'INSTALLATION

Durant l’installation de certains logiciels, ils vont vous demander si vous souhaitez installer d’autres logiciels, il est préférable de refuser l’installation de ces logiciels annexes. Ils peuvent contenir des virus ou d’autres programmes malveillants qui pourraient infecter votre ordinateur. Installer seulement ce dont vous avez besoin, quitte à installer l’autre logiciel d’une autre manière pour être sûr de l’origine.

LES MISES À JOUR

Il est important de faire les mises à jour de vos applications et de vos logiciels, ainsi que de votre ordinateur et de votre téléphone. Les mises à jour résolvent les problèmes, les bugs ainsi que les failles de sécurité. Les pirates deviennent de plus en plus rapides à trouver des failles et des vulnérabilités, il est donc important de faire les mises à jour quand elles sont disponibles.


Astuces : Dans les paramètres de vos logiciels, de votre ordinateur ou de votre téléphone vous pouvez rechercher s’il y a des mises à jour régulièrement. Pour les applications de votre téléphone, vous pouvez vérifier dans le Store d’applications pour les mises à jour.
